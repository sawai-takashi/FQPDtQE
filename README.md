## このリポジトリについて
- このリポジトリでは，論文「定性的な計画文書から量的評価へ：生成AIによる都市計画マスタープラン比較分析」における使用資料やファイルを公開している．

## 各ファイルについて
### analysisResult.csv
- prompt.mdをもとにGoogle Gemini 2.5 Proで評価した結果をまとめたものである．
- 各列の内容は以下の通り．
  - pref : 都県名(AI出力)
  - district : 自治体名(AI出力)
  - year : 発行年(AI出力．筆者によって確認し，修正が加えられているデータがある)
  - notmp : 総合振興計画フラグ(筆者記入．0ならば市町村MP，1ならば総合振興計画)
  - cluster : クラスタリング番号(筆者記入．エルボー法を用いたk = 4のk-meanクラスタリング値0 ~ 3を記入している）
  - sum : 1s ~ 9sの合計値(筆者記入)
  - 1-1score ~ 6-1score : それぞれ順番に評価指標1から9に対応するスコア値(AI出力)
    - prompt.mdも参考のこと
  - 1-1reason ~ 6-1reason : それぞれ順番に評価指標1から9に対応する根拠コメント(AI出力)

### masterPlanResources.csv
- 市町村マスタープランおよび総合振興計画の入手元についてまとめたcsvファイルである．
- 市区町村コードは総務省の全国地方公共団体コード(令和6年1月1日更新版)を利用している．
  - 入手元(2025/11/19アクセス) : https://www.soumu.go.jp/denshijiti/code.html
 
### prompt.md
- Gemini Pro 2.5に与えたプロンプト．各PDFごとの生成実行日はmasterPlanResources.csvにおけるアクセス日と同一である．
- 生成にあたってはGoogleのサービスである生成AIを用いたインターフェース生成アプリケーションサービスであるOpalを利用した．
  - https://opal.google
  - 事前のテストで，実行ごとに前回の生成内容(アクティビティ)に結果が誘引されないことを確かめている．
